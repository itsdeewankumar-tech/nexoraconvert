# 📖 About

NexoraConvert is a modern browser-based file conversion platform designed to make everyday file processing simple, fast, and secure.

The platform combines PDF tools, image utilities, and AI-powered features into a single responsive web application. Users can convert files without installing software, creating an account, or compromising privacy.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/NexoraConvert.git
```

---

## Navigate to Project

```bash
cd NexoraConvert
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

Open your browser:

```
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```
